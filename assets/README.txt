ASSETS FOLDER
=============

Drop these files here so the portfolio links work:

1. resume.pdf
   Your one-page resume. The "Download Resume" / "Download PDF" /
   "Print version" buttons all point to: assets/resume.pdf

2. (optional) photo.jpg
   Your professional photo. To use it, open index.html, find the
   <div class="photo-ph"> block in the About section, and replace the
   placeholder with:  <img src="assets/photo.jpg" alt="Your Name" />

3. (optional) project screenshots
   The project cards currently use hand-built CSS mockups (no images
   needed). If you'd rather show real screenshots, you can swap the
   .project-visual contents for <img> tags pointing here.
